## 一、诗人概况

&emsp;&emsp;张枣,巴蜀五君子之一，中国先锋诗歌的代表人之一张枣以“后赫耳墨斯学派（Posthermetische Schule）”闻名于世，被认为一方面给世界文学添加了创新的元素，另一方面又在作品中保留了中国古诗词的特点。著有《中国文化现代性研究》(德)，诗集《春秋来信》《张枣的诗》，以及随笔《张枣随笔选》，另有英语、德语诗歌和童话译作若干。
